# 📊 Cyber Attack Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents a **Cyber Attack Analysis Dashboard** built using Power BI.
It analyzes cyber attack data to identify patterns, trends, and potential security risks.

---

## 🎯 Objectives

* Monitor cyber attack activities
* Identify high-risk attack types
* Analyze trends over time
* Support decision-making for cybersecurity strategies

---

## 📊 Key Performance Indicators (KPIs)

* 🔹 Total Number of Attacks
* 🔹 Attack Types Distribution
* 🔹 Year-wise / Time-wise Attack Trends
* 🔹 Most Targeted Systems
* 🔹 High-Risk Attack Categories

---

## 📈 Insights

* 📌 Certain attack types occur more frequently than others
* 📌 Attack patterns show trends over time (increase/decrease)
* 📌 Specific systems or targets are more vulnerable
* 📌 Some periods experience higher attack volumes

---

## 💡 Business Impact

* Helps organizations **identify security vulnerabilities**
* Supports **better decision-making in cybersecurity planning**
* Enables **early detection of high-risk threats**
* Improves **overall system protection and response strategy**

---
